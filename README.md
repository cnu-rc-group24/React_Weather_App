# Group 24 React Native Weather App

충남대학교 실전코딩 React Native Weather App FE  

## How to run

### Clone the repository

```bash
$ git clone https://github.com/cnu-rc-group24/React_Weather_App
```

### Install whole dependencies

```bash
$ cd React_Weather_App
$ npm install
```

### Execute

```bash
$ npm run start
```
